## API Marvel

This app consumes a REST API, for more information visit the link: https://developer.marvel.com/docs


## Specifications and Technologies

- 100% Kotlin 
- Architecture MVVM
- Clean Architecture
- Design Patterns
- Data Binding
- Android Jetpack 
  * Navigation
  * Live Data
  * Room
- Data base SQlite
- Kotlin Coroutines
- Retrofit 2  
  * Interceptor OkHttp3
  * Converter Moshi
- Depedence Injection :syringe: Dagger 2
